Your First Java Program

Three Main Steps in Java Programming

  1. Write the program in Java and save it in a file.
  2. Compile the program to translate it from Java language into a special byte code that JVM understands
  3. Run the program.

Write a Programm

  • Open your notepad. Write a program follow the codes below:
    
public class HelloWorld {
    public static void main(string[] args) {
        System.out.println("Hello World, Command Line");
    }
}
  • Save file name
    Save file name as "helloworld.java" in to your working directory. For our case: C:\projects\javaprogram\
  • List Java Source Codes
    1. Open Command Line
    2. Navigate the java file by locating to "C:\projects\javaprogram\"
    3. Type: dir
    4. helloworld.java will be shown on the list
  • Compile Java Source Codes
    1. Type: javac helloworld.java
    2. Hit "Enter" kay
